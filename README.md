# FreshMart Grocery Sales Dashboard

## Project Overview

This project focuses on analyzing grocery sales data using Power BI to uncover insights related to sales performance, customer behavior, employee performance, and product categories.

The dashboard enables stakeholders to monitor key metrics and identify opportunities for business growth.

---

## Tools and Technologies

- Power BI
- DAX
- Excel

---

## Dashboard Pages

### Overview Dashboard
- Revenue KPIs
- Sales trends
- Country-wise performance

### Customer and Employee Analysis
- Customer behavior analysis
- Employee performance metrics

### Geographical Analysis
- City and country-wise sales analysis
- Regional performance comparison

### Product Category Analysis
- Product category performance
- Revenue contribution analysis

---

## Key Insights

- Identified top-performing products and categories.
- Analyzed customer purchasing behavior.
- Evaluated employee performance.
- Discovered geographical sales trends.
- Enabled data-driven decision-making through interactive dashboards.
